REPL Functionality

run(script, autoexec)

Runs the REPL in one of three modes, depending on the parameters.  The modes are as follows: script and autoexec both nil - interactive mode script not nil and autoexec nil - scripting mode script and autoexec not nil - autoexec mode

Parameters

scriptthe name of a script file
autoexeca flag, determining if autoexec mode is active
